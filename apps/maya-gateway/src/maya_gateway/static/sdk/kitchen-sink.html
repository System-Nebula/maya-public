Maya Voice SDK — Kitchen Sink

Drop-in Alpine.js components for the operator voice control panel. The full conversational pipeline is wired live: listen (detection engine) → reasoning model → Maya's turn. Settings persist to localStorage and all panels share one Alpine.store("mayaVoice").
